Project Information

This project is about developing a simple and secure miniFacebook web application.

Features

  • Account registration
  • Login
  • Change password
  • Edit profile
  • Create and edit posts
  • View and add comments on any post
  • Real-time chat with others
  • Superusers

Security

  • Deployed in HTTPS
  • Hashed passwords
  • SQL Prepared Statements
  • Input validation
  • HTML sanitization
  • Session authentication
  • CSRF attack protection
